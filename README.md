# Hyper Casual Wood Shop Game

## Overview
A **commercial hyper-casual mobile game** inspired by typical playable ad mechanics.  
The player chops trees, stacks wood on a conveyor belt, delivers it to a shop, and sells it to NPC customers.

The game includes a **simple upgrade shop** that allows players to improve production speed and overall efficiency.

---

## Core Gameplay
- Tree chopping
- Wood collection and stacking
- Conveyor belt delivery system
- NPC customers purchasing resources
- Upgrade shop for player progression

---

## Technical Stack
- **Unity**
- **Zenject** — Dependency Injection
- **UniRx** — Reactive gameplay logic and event handling
